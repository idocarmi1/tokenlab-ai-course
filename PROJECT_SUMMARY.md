# סיכום פרויקט להגשה

## שם הפרויקט

TokenLab — איך מודל שפה קורא משפט?

## שם הסטודנט

עידו כרמי

## המושג שנבחר

טוקנים במודלי שפה גדולים.

## שיעור בקורס

שיעור 9 — Language Models, Large Language Models, Next Token Prediction, Tokens.

## סוג אמצעי ההמחשה

אתר אינטראקטיבי בעברית ובכיוון RTL, המדגים באופן ויזואלי ופדגוגי כיצד מודל שפה גדול מעבד טקסט.

## מטרת הפרויקט

מטרת הפרויקט היא לעזור לסטודנטים עתידיים להבין את מושג הטוקנים במודלי שפה גדולים דרך התנסות פשוטה וברורה. האתר מחליף הסבר תיאורטי בלבד בהמחשה אינטראקטיבית שמראה כיצד משפט מתפרק ליחידות קטנות, כיצד היחידות מיוצגות במספרים, וכיצד המודל משתמש בהן כדי לנבא את ההמשך.

## מה האתר עושה

האתר מאפשר למשתמש להכניס משפט בעברית, לפרק אותו לטוקנים בצורה סימולטיבית, לראות Token ID מדומה לכל טוקן, לצפות בהדגמה של ניבוי הטוקן הבא באמצעות הסתברויות, ולענות על חידון קצר לבדיקת הבנה.

## למה בחרתי בנושא

בחרתי בנושא טוקנים מפני שזהו מושג בסיסי להבנת אופן הפעולה של כלי AI מודרניים כמו ChatGPT. בשיעור 9 למדנו שמודלי שפה גדולים מאומנים לחזות את הטוקן הבא ברצף, ולכן חשוב להבין איך הטקסט מתפרק לטוקנים לפני שהמודל יכול לבצע חישובים או ניבוי.

## איך האתר נבנה

האתר נבנה בעזרת React ו-Vite, עם CSS רגיל לעיצוב ממשק משתמש מודרני בעברית ובכיוון RTL. הפרויקט נבנה ללא שרת צד אחורי וללא שימוש ב-API חיצוני. במהלך הפיתוח נעשה שימוש בכלי AI וב-Codex לצורך כתיבה, שיפור ועיצוב האתר.

## הערה חשובה

האתר הוא סימולציה פדגוגית בלבד ואינו טוקנייזר אמיתי של OpenAI. ה-Token IDs והחלוקה לטוקנים באתר הם מדומים ונועדו להמחיש את הרעיון המרכזי בצורה פשוטה וברורה.
