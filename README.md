# TokenLab — איך מודל שפה קורא משפט?

## רקע כללי

הפרויקט נבנה במסגרת הקורס "יישומי AI בעולם העסקי" — תשפ"ו, סמסטר ב'.
מטרת העבודה היא לבחור מושג תיאורטי אחד מתוך הקורס ולבנות עבורו אמצעי המחשה פדגוגי שיעזור לסטודנטים עתידיים להבין את המושג בצורה פשוטה וברורה.

## המושג שנבחר

המושג שנבחר הוא:
טוקנים במודלי שפה גדולים.

המושג נלקח מתוך שיעור 9, שבו נלמדו נושאים כמו:

* Language Models
* Large Language Models
* פונקציית ניבוי המילה הבאה
* טוקנים במודלי שפה גדולים

## מטרת הפרויקט

מטרת הפרויקט היא להמחיש כיצד מודל שפה גדול, כמו ChatGPT, מעבד טקסט.
האתר מציג בצורה אינטראקטיבית את התהליך שבו משפט רגיל מתפרק לטוקנים, כל טוקן מקבל ייצוג מספרי, ולאחר מכן המודל משתמש ברצף הטוקנים כדי להעריך מה הטוקן הבא שהכי סביר שיופיע.

## למה בחרתי בנושא הזה

בחרתי בנושא טוקנים במודלי שפה גדולים מפני שזהו מושג בסיסי להבנת אופן הפעולה של כלי AI מודרניים.
בשיעור 9 למדנו שמודלי שפה גדולים מאומנים על כמויות עצומות של טקסטים, והמטרה המרכזית שלהם היא לחזות את המילה או הטוקן הבא ברצף.
כדי לעשות זאת, המודל צריך קודם לפרק את הטקסט ליחידות קטנות שנקראות טוקנים.

## מה האתר מדגים

האתר מציג את התהליך הבא:

1. המשתמש מכניס משפט בעברית.
2. האתר מפרק את המשפט לטוקנים בצורה סימולטיבית.
3. כל טוקן מקבל Token ID מספרי מדומה.
4. האתר מציג הדגמה של ניבוי הטוקן הבא באמצעות הסתברויות.
5. המשתמש עונה על חידון קצר כדי לבדוק הבנה.

## איך בנינו את הפרויקט

הפרויקט נבנה כאתר אינטראקטיבי בעזרת React ו-Vite.
נעשה שימוש ב-CSS לעיצוב ממשק משתמש בעברית ובכיוון RTL.
בנוסף, נעזרנו בכלי AI וב-Codex כדי לכתוב, לשפר ולעצב את האתר בצורה מקצועית וברורה.

האתר אינו משתמש בשרת צד אחורי, אינו משתמש ב-API חיצוני, ואינו שולח מידע החוצה.
כל ההדגמה מתבצעת בצד הלקוח בלבד.

## למה הפרויקט משמש

הפרויקט משמש כאמצעי המחשה לימודי.
הוא מיועד לעזור לסטודנטים להבין בצורה ויזואלית ואינטראקטיבית איך מודל שפה גדול עובד מאחורי הקלעים.

במקום לקרוא רק הסבר תיאורטי, המשתמש רואה בעצמו:

* איך משפט מתפרק לטוקנים
* איך טוקנים מקבלים ייצוג מספרי
* איך המודל משתמש בטוקנים כדי לנבא המשך אפשרי
* למה כמות הטוקנים חשובה גם בהקשרים עסקיים

## חשיבות עסקית

בעולם העסקי, הבנת טוקנים חשובה מפני שכמות הטוקנים יכולה להשפיע על:

* עלות השימוש במודלי AI
* אורך הקלט שניתן לשלוח למודל
* אורך התשובה שהמודל יכול להחזיר
* עבודה עם מסמכים ארוכים, חוזים, מיילים ופניות לקוחות
* תכנון נכון של צ'אטבוטים וכלי אוטומציה מבוססי AI

## הערה חשובה

האתר הוא סימולציה פדגוגית לצורכי למידה בלבד.
הוא אינו טוקנייזר אמיתי של OpenAI ואינו מציג Token IDs אמיתיים.
המטרה היא להמחיש את הרעיון המרכזי בצורה פשוטה וברורה.
